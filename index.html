Printable guitar tabs composer

Hi! This application allows you to generate guitar tabs that are properly prepared for printing. Although you need to take a moment to complete the tablature, after printing it will be super readable and fully useful. No more uneven bars and lopsided numbers. The application divides tablature by itself into pages, so you don't have to worry about anything. Save your favorite piece, print it and practice. Have fun!

Any help needed? Click here

1. At the beginning, you can enter the song title on top and add a short description below.
2. On the left side of the first tablature there is a standard tune presented for each string, but if you want you can change it.
3. To enter values ​​in the tab, simply click in the place of interest and enter the value using the keyboard. Each tablature allows you to enter 8 values ​​for each string.
4. If you need to add another tablature use the '+Add tab' button on the right. The '-Remove tab' button deletes the last visible tablature. The 'Print' button calls up the print window. The menu with buttons will appear after you will scroll down.
5. Questions or suggestions? contact@tabs4print.net
